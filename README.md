# Python Logic Lab 🧪

A structured environment for mastering core Python patterns and algorithmic logic. This repository serves as the foundation for building complex, stable automation frameworks.

## 🎯 Implementation Goals
- **Modularity:** Separation of orchestration logic (`logic_drills/`) and utility functions (`scripts/`).
- **State Management:** Handling persistent variables across loops.
- **Robustness:** Input sanitization and error handling to prevent runtime crashes.

## 🕹️ Current Drills
- **Number Guessing Simulator:** Implements `try-except` blocks for value validation and fixed-state target logic.
- **Modular Dice Simulator:** Uses functional decomposition to simulate probability outcomes.

## 🚀 How to Run
1. Clone: `git clone https://github.com/sahu-sourabh/python-logic-lab.git`
2. Execute: `python logic_drills/number_guess.py`
